# counter-program

## install solana

```sh
sh -c "$(curl -sSfL https://release.solana.com/v1.9.1/install)"
```

add path:

```sh
export PATH="/home/gitpod/.local/share/solana/install/active_release/bin:$PATH"
```

## install anchor

```sh
npm i -g @project-serum/anchor-cli
```

## init anchor project

```sh
anchor init <your-project-name>
```
